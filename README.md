## Description

Discover, connect and share. Empower your online presence with Magaaazine. With feature of sending messages, images, and files anonymously.

## Getting Started

### Dependencies

#### Front-End
* NextJS 13 (App Router).
* Zustand
* Tanstack React-Query
* Tailwind CSS

#### Back-End
* tRPC
* Prisma
* Supabase
* PostgreSQL

### Installing
```
pnpm i
```

### Executing program
```
pnpm dev
```
